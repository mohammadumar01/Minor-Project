## Description
This project focuses on modeling communication bottlenecks in High Performance Computing (HPC) applications using SST-based network simulation.

It analyzes MPI communication patterns of MiniFE and MiniMD applications and evaluates performance across multiple network topologies such as:

- Mesh
- Torus
- Dragonfly

The goal was to understand the impact of communication latency and network topology on large-scale parallel applications.
